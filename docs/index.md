# Network Labs - Cisco Training

Willkommen zu den Cisco Network Labs! Diese Sammlung enthält praxisorientierte Übungen für Cisco-Netzwerkgeräte.

## Verfügbare Labs

### VLAN Router-on-a-Stick Konfiguration

Eine umfassende Übung zur Implementierung von VLANs und Inter-VLAN Routing mit Router-on-a-Stick Technik.

**Schwerpunkte:**
- VLAN-Konfiguration auf Cisco Switches
- Trunk und Access Ports
- 802.1Q Subinterfaces auf Router
- Inter-VLAN Routing
- Netzwerk-Troubleshooting

**Ressourcen:**
- [Lab-Anleitung (Online)](labs/vlan-router-on-stick/index.md)
